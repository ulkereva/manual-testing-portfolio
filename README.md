# manual-testing-portfolio
Manual test cases and bug reports for Sauce Demo — a practice e-commerce web application.
What is Sauce Demo?
A publicly available test site simulating an online shop. Used by QA engineers worldwide to practice testing skills.

Tools Used
Browser: Chrome 124
Bug tracking: Jira (exported to Markdown for portfolio)
Test environment: https://www.saucedemo.com

How to Explore
Go to https://www.saucedemo.com
Use credentials: standard_user / secret_sauce
Follow the test steps in each .md file
